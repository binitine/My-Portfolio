# Images Folder

This folder contains all the images used in your portfolio website.

## Required Images

Please add the following images to this folder:

### 1. Hero Image
- **Filename**: `hero-image.jpg` (or `.png`, `.webp`)
- **Recommended size**: 500x500 pixels or larger (square or portrait)
- **Format**: JPG, PNG, or WebP
- **Usage**: Hero section (main landing area)
- **Note**: This image will have a floating animation effect

### 2. Profile Photo
- **Filename**: `profile.jpg` (or `.png`, `.webp`)
- **Recommended size**: 400x400 pixels (square)
- **Format**: JPG, PNG, or WebP
- **Usage**: About Me section

### 3. Project Screenshots
- **Project 1**: `project1.jpg` (or `.png`, `.webp`)
  - **Recommended size**: 600x400 pixels (16:9 aspect ratio)
  - **Usage**: First project in Projects Gallery

- **Project 2**: `project2.jpg` (or `.png`, `.webp`)
  - **Recommended size**: 600x400 pixels (16:9 aspect ratio)
  - **Usage**: Second project in Projects Gallery

- **Project 3**: `project3.jpg` (or `.png`, `.webp`)
  - **Recommended size**: 600x400 pixels (16:9 aspect ratio)
  - **Usage**: Third project in Projects Gallery

## Image Tips

1. **Optimize your images**: Use tools like TinyPNG or Squoosh to compress images before uploading
2. **File formats**: 
   - JPG: Best for photos
   - PNG: Best for screenshots with text
   - WebP: Best for modern browsers (smallest file size)
3. **Naming**: Keep filenames lowercase with no spaces (use hyphens or underscores)
4. **Alt text**: Make sure to update the `alt` attribute in HTML if you change image names

## How to Add Images

1. Place your image files in this `images` folder
2. Make sure the filenames match exactly:
   - `hero-image.jpg` (or your preferred format)
   - `profile.jpg` (or your preferred format)
   - `project1.jpg`
   - `project2.jpg`
   - `project3.jpg`
3. If you use different filenames, update the `src` attribute in `index.html`
